## Video Demo Recording Notes

**Slides 5-12 are video walkthroughs.** Record them as one continuous screen recording with these guidelines:

1. **Pre-load everything** — have the map, buildings, and data cached before recording.
2. **Move slowly and deliberately** — the audience needs time to read the UI.
3. **Mouse movements should be smooth** — no jerky clicking. Hover over elements before clicking.
4. **Record at 1920x1080** minimum (the projector at City Hall is likely 16:9).
5. **Have a backup recording** embedded in the PowerPoint in case live demo fails.
6. **Practice the demo flow at least 5 times** — every click should be muscle memory.

**Demo flow order:**
Map overview (Slide 5) → Building Editor + Voice (Slide 6) → Tree Advisor (Slide 7) → Export to Map → Environmental Impact (Slide 8) → Shadow Analysis (Slide 9) → Zoning (Slide 10) → Traffic/Stakeholder/Drainage (Slide 11)
